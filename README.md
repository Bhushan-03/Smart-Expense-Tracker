# Smart Expense Tracker

## Project Description:
Smart Expense Tracker is a Python-based CLI application that helps users to track, manage, analyze their expenses. 
It allows users to add, view, update, delete, analyze and export their expense summary efficiently.


## Features:
- Add / View / Update / Delete Expenses
- SQLite Database Storage
- SQL Filtering
- Keyword Search
- Expense Analytics Dashboard
- CSV Export
- TXT Report Export
- Database Backup
- Table Display


## Tech Stack:
- Python
- SQLite (Data Storage)
- Tabulate Library
- CSV Module
- Datetime module
- Logging


## Installation:
Install required library:
- pip install tabulate
