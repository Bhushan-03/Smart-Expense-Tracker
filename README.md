# Smart Expense Tracker

## Project Description:
Smart Expense Tracker is a Python-based CLI application that helps users to track, manage, analyze their expenses. 
It allows users to add, view, update, delete, analyze and export their expense summary efficiently.


## Features:
- Add Expense
- View Expense
- Update Expense
- Delete Expense
- Expense Analytics
- Monthly Summary
- Category Summary
- Export Report (TXT & CSV)
- Table UI
- Error Logging System
- Data Storage (Json)


## Tech Stack:
- Python
- SQLite (Data Storage)
- Tabulate Library
- CSV Module
- Datetime module


## Installation:
Install required library:
- pip install tabulate

## How to run:
file to run:
